# CodexBridge Connector

One-shot bootstrap scripts for connecting a personal computer to CodexBridge mobile app.

## Files

- `connect.sh` for macOS/Linux terminal
- `connect.ps1` for Windows PowerShell
- `SHA256SUMS` checksums for the scripts
- `SHA256SUMS.sig` detached signature for checksum file (`ssh-keygen -Y`)
- `SIGNING_KEY.pub` public signing key

## Security model

1. Mobile app copies a bootstrap command.
2. Command downloads `connect.*`, `SHA256SUMS`, and `SHA256SUMS.sig`.
3. Command verifies signature of `SHA256SUMS` using the embedded signing public key.
4. Command verifies script SHA256 against trusted checksums.
5. Script executes only if both checks pass.

## Usage (generated by app)

Use commands generated in app (they already include signature + SHA256 checks).

## Manual verification

```bash
printf 'codexbridge %s\n' "$(awk '{print $1" "$2}' SIGNING_KEY.pub)" > /tmp/allowed_signers
ssh-keygen -Y verify -f /tmp/allowed_signers -I codexbridge -n file -s SHA256SUMS.sig < SHA256SUMS
shasum -a 256 -c SHA256SUMS
```

## Requirements

- `codex` installed and authenticated on the computer.
- `ssh-keygen` available (OpenSSH client).
- Either `cloudflared` or `ngrok` installed.
